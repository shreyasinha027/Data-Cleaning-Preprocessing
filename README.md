# DS Job Data Cleaning & Preprocessing

This project demonstrates data cleaning and preprocessing techniques for Machine Learning using Python, Pandas, NumPy, and Scikit-Learn.

## Features

- Data exploration
- Missing value handling
- Duplicate removal
- Feature engineering
- Salary processing
- Outlier detection
- One-Hot Encoding
- Feature Scaling

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

## Project Structure

```
DS-Job-Data-Cleaning/
│
├── data_cleaning.py
├── README.md
├── requirements.txt
└── Uncleaned_DS_jobs.csv.zip
```

## Run

```bash
pip install -r requirements.txt
python data_cleaning.py
```
